# Nitrogen-Corn Data

Daily alternative data feeds for the Nitrogen-Corn QuantConnect strategy.

## Files
- `alternative/satellite_ndvi_history.csv` — weather-derived NDVI proxy
- `alternative/nola_logistics_history.csv` — Mississippi River gauge height
- `modeling/golden_weights.csv` — optimized signal weights

Updated daily by `run_pipeline.py` in the strategy repo.
